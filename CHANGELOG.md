# Changelog

## [0.1.1] - 2026-01-20

### Fixed
- Fixed lowercase SVG elements (g, rect, defs, clipPath) causing React Native errors
- Updated 16 icons to use capitalized components (G, Rect, Defs, ClipPath)

## [0.1.0] - 2026-01-20

### Added
- Initial release with 129 icons for React Native/Expo
- TypeScript support with full type definitions
- Compatible with react-native-svg (>=13.0.0)
- Customizable size, color, and strokeWidth props
